# Cidadania-Digital-e-Intelig-ncia-Artificial# Cidadania Digital e Inteligência Artificial
## O Impacto das Deepfakes e da Desinformação na Sociedade

### 🎯 Objetivo do Site
O objetivo deste site é conscientizar a comunidade escolar sobre os perigos das mídias manipuladas (Deepfakes) e notícias falsas automatizadas através de uma dinâmica interativa de Jogo da Memória.

### 🤖 Declaração de Uso de IA (Créditos)
* **Texto e Código Base:** Gerados com auxílio do modelo Gemini da Google.
* **Prompt utilizado:** "preciso criar um site com um jogo da memoria seguindo o: [texto completo do enunciado fornecido]"
